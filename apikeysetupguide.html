How to set up Google API key

1. Go to Google Cloud Console

2. Click the project drop-down arrow, select or create the project you would like to make the key in.

3. Click the Navigation Menu on the top left corner, and select APIs & Services, then select Credentials.

4. Click Create Credentials on the top, then select API key.

5. Copy the key, this will be the value to be entered into the input form.

6. Click the Navigation Menu button on the top left corner, and select APIs & Services, then select Library.

7. Search for Geocoding API, then click Enable button.

Billing

1. Fill out the form at Billing From.

2. Go to Google Console Bill page, and select the project with your geocoding API. Then select the billing account that you have just set up.
